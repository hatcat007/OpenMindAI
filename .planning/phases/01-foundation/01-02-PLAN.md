<task type="auto">
  <name>Task 2: Create Plugin Implementation</name>
  <files>src/plugin.ts</files>
  <action>
    Create the core plugin implementation using @opencode-ai/plugin SDK.
    
    **CRITICAL:** Storage is SYNCHRONOUS (bun:sqlite design). No await needed for storage operations.
    
    Import from @opencode-ai/plugin:
    ```typescript
    import type { Plugin } from "@opencode-ai/plugin";
    ```
    
    **CRITICAL:** Use context destructuring pattern:
    ```typescript
    // CORRECT: Destructure from context object
    export const OpencodeBrainPlugin: Plugin = async ({ client, directory, worktree, $ }) => {
      // ...
    };
    
    // WRONG: Don't treat ctx as single param
    // export const OpencodeBrainPlugin: Plugin = async (ctx) => { ... } // ctx is the context object!
    ```
    
    Implement plugin function:
    ```typescript
    export const OpencodeBrainPlugin: Plugin = async ({ client, directory, worktree }) => {
      // Load configuration
      const config = loadConfig({ directory, worktree });
      const storagePath = getStoragePath(worktree || directory, config);
      
      // Initialize storage SYNCHRONOUSLY
      const storage = createStorage({ filePath: storagePath }); // NO await!
      
      return {
        // Session start
        "session.created": async ({ session }) => {
          if (config.debug) {
            client.app.log({
              service: "opencode-brain",
              level: "info",
              message: `Storage initialized for session ${session.id}`,
            });
          }
        },
        
        // Tool execution - capture for future phases
        "tool.execute.after": async (input) => {
          // Stub for Phase 2 - just log if debug mode
          if (config.debug) {
            console.log("[opencode-brain] Tool executed:", input.tool);
          }
        },
        
        // Session end - close storage cleanly
        "session.deleted": async () => {
          // Use session.deleted (not session.idle) for cleanup
          if (config.debug) {
            console.log("[opencode-brain] Session ended, closing storage");
          }
          storage.close(); // SYNCHRONOUS - no await
        },
        
        // Handle errors gracefully
        onError: (error) => {
          console.error("[opencode-brain] Plugin error:", error.message);
          // Don't re-throw - keep Opencode running
        }
      };
    };
    ```
    
    Key requirements:
    1. **Context destructuring:** Use `async ({ client, worktree, directory })` pattern
    2. **Auto-initialization:** Storage creates file automatically if missing
    3. **Worktree detection:** Use worktree || directory for project-specific storage
    4. **Error handling:** Never throw from event handlers - log and continue
    5. **Debug mode:** Use client.app.log() for proper Opencode logging
    6. **Clean shutdown:** Use session.deleted (not idle) + synchronous close()
    
    Import storage layer from Plan 01:
    ```typescript
    import { createStorage } from "./storage/sqlite-storage.js";
    ```
    
    Add extensive JSDoc explaining:
    - Plugin lifecycle (created → session.created → session.deleted)
    - Context destructuring pattern
    - Synchronous storage design
